Qlip とは
🚀

素早い起動

カードをクリック、またはキーワードを入力して Enter するだけで、ファイル・フォルダ・Web サイト・ツールを即座に開けます。

🗂

カテゴリ整理

お気に入り・PJフォルダ・ツール・書類など、カテゴリごとにカードを整理。サイドバーでワンクリック切り替え。

🔍

インクリメンタル検索

名前・パス・説明文でリアルタイム絞り込み。1件一致で即起動、複数一致でグリッド絞り込み表示。

🔀

D&Dで並び替え NEW

カードを0.5秒長押しするとゴーストが表示され、そのままドラッグ＆ドロップで好きな位置に移動できます。

対応する起動タイプ： ファイル（.exe / .xlsm / .txt など）　／　フォルダ（ローカル・ネットワーク共有）　／　Web URL（http / https）　／　PowerShell スクリプト（.ps1）　／　バッチファイル（.bat / .cmd）

ショートカットの起動方法
🖱

カードをクリック

グリッド上のカードを左クリックするだけで即座に起動します。

⌨

名前で検索 → Enter

検索バーに名前を入力して Enter。1件一致なら即起動、複数一致なら絞り込み表示します。

🔗

パス・説明でも検索可

パスの一部や説明文のキーワードでも検索できます。

カードの並び替え（ドラッグ＆ドロップ） v1.1 NEW
1

移動したいカードを 0.5秒 長押し

カード上でマウスボタンを押したまま約0.5秒待つと、青いゴーストラベル（✥ 名前）が表示されてD&Dモードになります。

2

移動先へドラッグ

ゴーストを移動先のセルまでドラッグします。グリッド上どこでも移動可能です。

3

マウスボタンを離すと確定

ドロップした位置にカードが移動し、順序が qlip_data.json に即座に保存されます。

ポイント： 短いクリック（0.5秒未満）は通常の起動として動作します。D&Dと起動は自動的に区別されるので、誤操作の心配はありません。
🖼
カード内画像表示 v1.1 NEW

ショートカットに画像を設定すると、カード内にサムネイル（最大 96×54px）が表示されます。視覚的にツールを識別しやすくなります。

🖼

画像の設定方法

「✎ 編集」ダイアログの「画像」欄に PNG ファイルのパスを指定します。「選択」ボタンでファイルダイアログから選択可能。

🔲

表示の ON/OFF

左サイドバー下部の「設定」セクションにあるトグルスイッチで、全カードの画像表示を一括で切り替えできます。

💾

設定は自動保存

ON/OFF の状態は qlip_data.json に保存され、次回起動時も維持されます。

✏
ショートカットの追加・編集・削除
1

「＋ 追加」または「✎ 編集」をクリック

サイドバー下部のボタンをクリックすると、ダイアログが開きます。

2

各項目を入力

カテゴリ・名前・パス は必須です。
「参照」ボタンでファイル／フォルダをダイアログから選択できます。

3

説明・画像・枠色を任意設定

説明：プレビューパネルに表示するテキストメモ。
画像：カード内に表示するサムネイル（PNG/GIF）。
枠色：カードの枠色を HEX カラーで指定（空欄でカテゴリ色を使用）。

4

「追加」または「更新」をクリックして保存

qlip_data.json に自動保存されます。

—

削除する場合

編集ダイアログで対象を選択し、「削除」ボタンをクリックして確認します。

💻 RDP専用の追加・編集： サイドバーの「💻 RDP」カテゴリ以下には、RDP接続専用の「＋ RDP追加」「✎ RDP編集」ボタンが用意されています。RDP接続先（IP、ユーザー名、パスワード等）はこちらの専用機能から管理・接続できます。

よくある操作・Tips
⚡

即時起動（検索）

アプリ起動直後から検索バーにフォーカスがあります。そのままショートカット名を入力して Enter を押すだけ。

🗂

カテゴリ切り替え

サイドバーのカテゴリ名をクリックすると、そのカテゴリのカードのみ表示されます。「All」で全件表示に戻ります。

🔀

カード並び替え

カードを0.5秒長押し → ゴースト表示 → ドラッグ → 離す。順序は即座に保存されます。

⌨️

RDP / お気に入り切替

Ctrl+R キーを押すことで、「RDP」カテゴリと「お気に入り（最初のカテゴリ）」を瞬時にトグル切り替えできます。

🖼

カード画像の設定

編集ダイアログの「画像」欄に PNG ファイルのパスを指定。images\ フォルダに置くと管理しやすいです。

🔲

画像表示の切り替え

左サイドバーの「設定」→「カード画像」トグルで全カードの画像表示を一括ON/OFF。設定は自動保存されます。

📊

ログの確認

Qlip.log に起動・検索・追加・削除・D&D移動の操作ログが記録されます。トラブル時に参照してください。
