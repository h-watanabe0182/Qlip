📋
Qlip とは
🚀

素早い起動

カードをクリック、またはキーワードを入力して Enter するだけで、ファイル・フォルダ・Web サイト・ツールを即座に開けます。

🗂

カテゴリ整理

お気に入り・PJフォルダ・ツール・書類など、カテゴリごとにカードを整理。サイドバーでワンクリック切り替え。

🔍

インクリメンタル検索

名前・パス・説明文でリアルタイム絞り込み。1件一致で即起動、複数一致でグリッド絞り込み表示。

🖼

プレビュー表示

カードにマウスを乗せると、左パネルにアイコン画像・説明文・パスをリアルタイムプレビュー。

対応する起動タイプ： ファイル（.exe / .xlsm / .txt など）　／　フォルダ（ローカル・ネットワーク共有）　／　Web URL（http / https）　／　PowerShell スクリプト（.ps1）　／　バッチファイル（.bat / .cmd）

ショートカットの起動方法
🖱

カードをクリック

グリッド上のカードを左クリックするだけで即座に起動します。

⌨

名前で検索 → Enter

検索バーに名前を入力して Enter。1件一致なら即起動、複数一致なら絞り込み表示します。

🔗

パス・説明でも検索可

パスの一部や説明文のキーワードでも検索できます。

タイプ起動方法特記事項
🌐 URL既定ブラウザで開くhttp:// または https:// で始まるもの
📄 ファイル関連付けアプリで開く.exe / .xlsx / .txt など
📁 フォルダエクスプローラーで開くネットワークパス（\\server\share）も可
⚡ .ps1新しいコンソール窓で PowerShell 実行ExecutionPolicy Bypass・NoExit で実行
📜 .bat / .cmd新しいコンソール窓で cmd 実行—
📝
アクセスリスト（設定ファイル）

Qlip は exe と同じフォルダに置かれた .txt ファイル（例：アクセスリスト.txt）を自動認識して読み込みます。 テキストエディタで直接編集することも、アプリ内の「＋ 追加」「✎ 編集」から操作することも可能です。

アクセスリスト.txt — 書式サンプル
# CATEGORY 0 お気に入り
# CATEGORY 1 PJフォルダ
# CATEGORY 2 ツール

0 mermaid C:\claudecode\Mermaid\index.html
0 todo http://172.17.2.205:5001/
1 kl \\server\projects\TEC\kl\01_DOC
2 tool C:\Tools
書式説明例
# CATEGORY <番号> <名前> カテゴリ定義行 # CATEGORY 0 お気に入り
<番号> <名前> <パス> ショートカット行（半角スペース区切り） 0 memo C:\Desktop\memo.txt
⚠ 注意： 名前に半角スペースは使用できません。アンダースコアやキャメルケースを推奨します。
ファイルは Shift-JIS（cp932）または UTF-8 の両方に対応しています。保存時は cp932 で書き込まれます。
✏
ショートカットの追加・編集・削除
1

「＋ 追加」または「✎ 編集」をクリック

サイドバー下部のボタンをクリックすると、ダイアログが開きます。

2

各項目を入力

カテゴリ・名前・パス は必須です。
「参照」ボタンでファイル／フォルダをダイアログから選択できます。

3

説明・画像・枠色を任意設定

説明：プレビューパネルに表示するテキストメモ。
画像：プレビューに表示するサムネイル（PNG/GIF）。
枠色：カードの枠色を HEX カラーで指定（空欄でカテゴリ色を使用）。

4

「追加」または「更新」をクリックして保存

アクセスリスト.txt と metadata.json が自動保存されます。

—

削除する場合

編集ダイアログで対象を選択し、「削除」ボタンをクリックして確認します。

💡
よくある操作・Tips
⚡

即時起動（検索）

アプリ起動直後にすでに検索バーにフォーカスがあります。そのままショートカット名を入力して Enter を押すだけ。

🗂

カテゴリ切り替え

サイドバーのカテゴリ名をクリックすると、そのカテゴリのカードのみ表示されます。「All」で全件表示に戻ります。

📋

テキスト直接編集

アクセスリスト.txt をテキストエディタで編集してアプリを再起動すると、変更が反映されます。

🖼

プレビュー画像の設定

編集ダイアログの「画像」欄に PNG ファイルのパスを指定します。images\ フォルダに置くと管理しやすいです。

🔄

リストの更新を反映

アクセスリスト.txt を外部で編集した場合は、Qlip を再起動すると最新の内容が読み込まれます。

📊

ログの確認

Qlip.log に起動・検索・追加・削除の操作ログが記録されます。トラブル時に参照してください。
